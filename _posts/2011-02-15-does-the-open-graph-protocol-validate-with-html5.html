--- tags: - html5 - opengraphprotocol - validation - Words published: true type: post status: publish title: Does the Open Graph Protocol validate with HTML5? layout: post meta: aktt_notify_twitter: "yes" aktt_tweeted: "1" _wp_old_slug: "" _edit_last: "1" --- TL;DR: Not yet. As far as I can gather, the closest you can get is: <!DOCTYPE html> <html version="HTML+RDFa 1.0"> <head> <meta charset="utf-8"> <meta property="og:title" content="Test"> <title>Test</title> </head> <body> <p>Test</p> </body> </html>

NB. The HTML+RDFa spec is still a working draft, and the W3C HTML5 validator is still experimental. Therefore this will not pass the W3C validator. But this is valid, according to the current spec.
